### MWCOG Example

This is an example of the Metropolitan Washington Council of Governments ActivitySim model. This is a 53 zone portion of the model, centered on the National Mall.

This is occasionally updated with the MWCOG Gen3 Model, last update 5/15/2023. Questions can be directed to Mark Moran (mmoran@mwcog.org), Feng Xie (fxie@mwcog.org), Ray Ngo (rngo@mwcog.org). 

Run with `python simulation.py -c configs_mp -c configs -d data -o output`
