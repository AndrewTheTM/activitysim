### MWCOG Example

This is an example of the Metropolitan Washington Council of Governments ActivitySim model. This is a 53 zone portion 
of the model, centered around the National Mall area.

This model features a work from home model, telecommute frequency model, and a transit pass subsidy model. The model
uses one zone system and uses Cube Voyager for auxiliary model steps and  network processing, skimming, and assignment 
steps.

The full model can be found at https://github.com/MWCOG/Gen3_Model.
